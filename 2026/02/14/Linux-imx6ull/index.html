虚拟机

嵌入式Linux开发需要在linux环境下进行，如果安装双系统那么一次只能用一个系统，因此需要下载虚拟机。
VMware是收费软件需要密钥，可以在tb上花几块钱买一个。

配置虚拟机

  • CPU分配（核心数量），一般55分
  • 内存分配 8G+
  • 硬盘 创建独立分区 300G+

新建虚拟机

选择Ubuntu 64位
在虚拟机创建上网课讲的非常啰嗦还是错的，我的笔记本使用i5-13500HX，如果未来要用ros这种编译比较多的可以给一个处理器*6核，这个根据自己电脑配置来定去问ai~

  • 硬盘 300G(其实60G就够)
  • CPU 1*8核
  • 内存 8G
  • 网络选择桥接模式
  • 其他默认

Ubuntu操作系统安装

安装好ubuntu系统后需要安装VMware tools，官方视频太古老了不好用，因为新版本的Ubuntu有open-vm-tools。我们打开终端，进入root。

现在就可以在虚拟机和win系统之间自由移动了，最重要的是可以复制粘贴两个系统的东西了。

Linux常用命令

目录操作

命令 说明
ls -a 查看目录所有文件和文件夹（包括隐藏文件）
cd / 进入 Linux 根目录
mkdir 创建文件夹

文件操作

命令 说明
touch 创建文件
cat 显示文件内容
cp 文件拷贝命令
mv a b 重命名文件（a → b）
mv a b/ 移动文件（a 移动到 b 文件夹里）
rm 删除文件
rm -r 删除目录（递归删除）
rm -f 强制删除（无需确认）
file 文件名 文件类型查看命令

查找搜索

命令 说明
find -name 文件名 按文件名查找
grep nr "ubuntu" / 在根目录递归查找包含”ubuntu”的内容（n显示行号，r递归）

系统信息

命令 说明
uname 系统信息查看命令
ps -aux 查看当前系统进程（类似任务管理器）
top 进程实时运行状态查看命令

网络相关

命令 说明
sudo apt install net-tools 安装网络工具包
ifconfig 显示网络信息（例如 IP 地址）

其他常用

命令 说明
clear 清屏
sync 数据同步写入磁盘

ubuntu文件系统结构

根目录是/。在嵌入式开发中知道这些文件都是干啥的还是挺重要的，不然有可能出现下好了驱动但是找不到的尴尬情况。。。

目录 作用
/bin 存放二进制可执行文件
/boot Ubuntu内核和启动文件
/dev 设备驱动文件
/etc 通常是一些配置文件（密码文件，profile设置环境变量）
/home 用户主文件夹
/lib & /lib64 库文件
/media 访问U盘
/proc 虚拟文件系统，以文件形式展示系统和进程的运行时信息。比如：
• /proc/cpuinfo → 查看CPU信息
• /proc/meminfo → 查看内存使用
• /proc/version → 查看内核版本
• /proc/uptime → 查看系统运行了多久
/tmp 临时文件
/usr unix software resource
/var 一些日志文件

绝对路径和相对路径

这个在Linux操作里也是经常要用，记住绝对路径是/根目录算起的路径；相对路径./是当前路径，../是上一层目录；../../../就是返回好几层。
